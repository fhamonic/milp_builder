# Solver_Builder

Library attempting to provide a nice API for at least GUROBI and Cbc solver.

## Dependencies

### Build process
The build process requires CMake 3.12 (https://cmake.org/) or more and the Conan C++ package manager (https://conan.io/).

## How to Compile
Just

    make
